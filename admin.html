1 How To Update Lens Pricing

When prices change (1-2 times per year), here is what to do:

1. Open your data.js file in Notepad on your computer
2. Find the lens you want to update
3. Change the number next to the price field
4. Update the LAST_UPDATED date at the top
5. Save the file
6. Come to this admin page and use the uploader below
7. Or go to GitHub directly and re-upload the data.js file

To add a new lens: Copy one of the existing lens blocks in data.js, paste it at the bottom of the LENS_DATA list, and fill in the new details. Make sure to add a comma after the previous last lens block.

2 Quick Lens Data Editor

You can edit the lens data directly below. When done click "Validate & Download" to save a new data.js file, then upload it to GitHub.

3 After Downloading — Upload To GitHub

After you download the updated data.js file:

1. Go to github.com and log in
2. Open your contact-lens-calculator repository
3. Click Add file → Upload files
4. Upload the new data.js file
5. In the commit message box type: Update lens pricing
6. Click Commit changes
